# Web Scraping Python Project

[ CLICK THIS -------> View the Interactive Project Dashboard](https://razaali901.github.io/Web-Scraping-Python-Project/)


A structured collection of real-world scrapers targeting **Dawn News**, **PSX**, **QS Rankings**, **Daraz**, and **Goodreads** — with automated report exports.

## 📊 Project Stats
* **5** Scrapers
* **8+** CSV/JSON outputs
* **10+** PDF reports
* **100%** Python

## 🛠 Tech Stack
* **Python 3**: Core language
* **BeautifulSoup4**: HTML parsing
* **Selenium**: JS-rendered sites
* **Requests**: HTTP client
* **ReportLab**: PDF generation

## 🚀 Tasks Overview
1.  **Dawn Headlines**: Scrapes top 30 headlines from `dawn.com`.
2.  **PSX Indices**: Real-time market summary from `psx.com.pk`.
3.  **QS Rankings**: World university rankings from `topuniversities.com`.
4.  **Daraz Listings**: Product data (e.g., iPhone 15) from `daraz.pk`.
5.  **Goodreads Books**: Top-rated books and reviews from `goodreads.com`.
